# Persona C2c — In-Process Switching Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Make one long-lived `HarnessAgent` process serve N persona sessions and switch between them in-process (no re-exec), each persona keeping its own session, memory, and model.

**Architecture:** Add an agent-owned seat map (`persona_id → Seat{session_id, model}`) plus an explicit `_active_persona`. A new `harness/set_persona` ACP ext-method get-or-creates a persona's seat, makes it active, and returns its session_id+model; the TUI repoints `self._session_id` and updates the indicator from the response. Per-seat model is resolved once at seat creation via a shared `resolve_session_model` (the launch path uses it too). The reducer stops renaming agent ids in place.

**Tech Stack:** Python 3.11, Textual TUI, the ACP (agent-client-protocol) lib, pytest. Worktree: `.claude/worktrees/persona-c2c` (branch `persona-c2c`).

**Source spec:** `docs/superpowers/specs/2026-06-27-persona-C2c-design.md` (revised, Codex-reviewed).

## Global Constraints

- **No re-exec for switching** — in-process routing only.
- **Model single-homed in `done.conf [agents.<id>]`** — resolve per-seat at seat creation; never add a second model home. `persona.toml` stays non-model.
- **No per-persona code BRANCH** — `"default"` is just persona id #0; no `if persona_id == "default":` in routing/model paths.
- **No-op guarantee** — no `--persona` + no persona files → one `"default"` seat (workspace = `default_workspace_dir()`, not literal `None`), engine-default model, inert templates, byte-identical to today.
- **Reuse before invent** (`harness/tui/styles/components.md`) — zero new TUI widgets; wire the shipped `AgentRail`.
- **Persona + memory resolve from `state.workspace_dir`** (Phase B invariant), now per-seat.
- **Work in this worktree only**; run pytest with the worktree as cwd (editable-install shadowing). Test command: `.venv/bin/python -m pytest tests/ -q` from the worktree root. Verify a surprising import with `python -c "import harness.acp_agent as m; print(m.__file__)"`.
- **The model ladder is `harness/model_resolve.resolve_model`** — a pure function already; do NOT re-implement the precedence. Rungs (highest first): `shell_env`, `persisted`, `dotenv`, `engine_default`; empty string counts as absent.

```python
# harness/persona_sessions.py
"""Per-persona seats for the in-process fleet: each persona id maps to a Seat
(its session_id + its resolved worker model). resolve_session_model applies the
C1 precedence ladder (harness.model_resolve.resolve_model) keyed on the persona's
done.conf entry, so every seat — launch persona included — resolves its model the
SAME way. No second model home: persisted model lives only in done.conf."""

from __future__ import annotations

from dataclasses import dataclass

from harness import config, model_resolve, vibeproxy


@dataclass(frozen=True)
class Seat:
    session_id: str
    model: str | None        # None in mock mode


def resolve_session_model(
    persona_id: str, *, shell_set_model: bool, shell_env: str | None,
    dotenv: str | None, backend: str,
) -> str | None:
    """The worker model for one persona's seat. None for mock. Otherwise the C1
    ladder: a REAL shell VIBEPROXY_MODEL (shell_set_model) forces the value for
    every persona; else this persona's done.conf model; else a .env value; else
    the engine default. shell_env/dotenv are the caller-read env values (kept as
    params so this stays pure-ish and testable)."""
    if backend == "mock":
        return None
    persisted = None
    cfg = config.load_agent(persona_id)
    if cfg is not None:
        persisted = cfg.model
    return model_resolve.resolve_model(
        shell_env=shell_env if shell_set_model else None,
        persisted=persisted,
        dotenv=dotenv,
        engine_default=vibeproxy.DEFAULT_MODEL,
    )


class PersonaSessions:
    """Agent-owned seat map: persona_id -> Seat. get_or_create resumes an existing
    seat (same session AND model) or mints+resolves a new one."""

    def __init__(self) -> None:
        self._seats: dict[str, Seat] = {}

    def seat_of(self, persona_id: str) -> Seat | None:
        return self._seats.get(persona_id)

    def model_of(self, persona_id: str) -> str | None:
        seat = self._seats.get(persona_id)
        return seat.model if seat is not None else None

    def set_model(self, persona_id: str, model: str | None) -> None:
        seat = self._seats.get(persona_id)
        if seat is not None:
            self._seats[persona_id] = Seat(session_id=seat.session_id, model=model)

    def get_or_create(self, persona_id, *, cwd, store, resolve_ws, resolve_model) -> Seat:
        seat = self._seats.get(persona_id)
        if seat is not None:
            return seat                      # resume: same session + same model
        ws = resolve_ws(persona_id)
        session_id = store.new(cwd=cwd, workspace_dir=ws)
        model = resolve_model(persona_id)
        seat = Seat(session_id=session_id, model=model)
        self._seats[persona_id] = seat
        return seat
```
